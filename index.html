Panduan Pendaftaran Untuk Aplikasi Transaksi Pulsa

Pertama silahkan download Aplikasi dengan mengklik tombol Download di bawah.

Download Aplikasi

1. Proses Pendaftaran

Silahkan install dan buka Aplikasi. Berikut tampilan awal aplikasi.

Silahkan klik DAFTAR.

Isi username dengan huruf kecil tanpa spasi dan harus unik.
Isi Nama dengan nama outlet atau nama sendiri.
Masukan Password minimal 8 huruf kombinasi huruf dan angka.
Masukan ulang Password diatas.

Jika berhasil akan muncul popup pemberitahuan. Klik OK untuk melanjutkan.
Mohon diingat username yang dilingkari merah.

2. Pararel Aplikasi

Jika sudah berhasil mendaftar silahkan lakukan pararel.
Untuk Melakukan pararel bisa hubungi CS via whatsapp dengan menyertakan username yang telah didaftarkan.

Atau bisa melakukan pararel sendiri dengan mengirmkan format berikut ke center WA atau SMS:
PAR.@username.PIN
Contoh:PAR.@salmapulsa.1234
berikut jika pararel berhasil dilakukan:

Proses pararel telah berhasil. Sekarang kamu sudah bisa menggunakan aplikasi untuk transksi.
